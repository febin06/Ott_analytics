### **OTT Analytics Dashboard (Excel)**





#### **Project Overview**



This project is an end-to-end Excel-based analytics dashboard built using OTT platform user data from 2023 to 2025.  

The goal is to analyze user churn, revenue contribution, device usage, and content engagement and present insights in a clear, business-friendly dashboard.



This project demonstrates data cleaning, pivot analysis, KPI tracking, and dashboard design using Microsoft Excel.





##### **Business Questions Addressed**



\- What is the overall churn rate of users?

\- How does churn trend month-over-month?

\- Which subscription plan contributes the highest revenue?

\- Which device types experience higher churn?

\- Which content genres generate the highest watch hours?





#### **Tools \& Techniques Used**



\- Microsoft Excel

\- Data Cleaning \& Transformation

\- Pivot Tables \& Pivot Charts

\- KPI Cards

\- Line, Bar \& Donut Charts

\- Dashboard Layout \& Visual Design





##### **Key KPIs**



\- Total Users

\- Total Active Users

\- Total Churned Users

\- Churn Rate

\- Average Revenue Per User (ARPU)







#### **Dashboard Insights**



\- Peak churn occurred in September 2024

\- Standard subscription plan contributes the highest revenue (~40%)

\- Mobile users show the highest churn among devices

\- Action and Crime genres dominate total watch hours

\- Premium users show lower churn, indicating stronger customer loyalty
